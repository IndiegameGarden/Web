What is Indiegame Garden?
System requirements
  • Windows XP or Windows 7 gaming PC with faily modern graphics card
  • Fast Internet connection (for the automatic game downloads)
  • For Windows 7 you may need to install the  Microsoft XNA 4.0 redistributable. (You can first try without installing this.)
  • For Windows XP you may need to install the  Microsoft XNA 4.0 redistributable, DotNet 4.0 or in rare cases DirectX. (You can first try without installing this.)
Download
For Windows XP / Windows 7 download the "Portable app" in the .zip file here. (Version Alpha-5 2012-06-12)
Play
Unzip the contents to anywhere you like, for example to Desktop or the C:\ drive. Then double-click IndiegameGarden.exe. (It will warn you twice that the application is not signed.)
Note: games that you 'grow' in your garden are all stored in the folder IndiegameGarden_data.
