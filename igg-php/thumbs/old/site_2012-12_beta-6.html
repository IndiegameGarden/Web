What is Indiegame Garden?
"A pleasant free game downloader" according to the review at DIYGamer.com. It's an app store for free Windows games. It's a garden, where indie games grow! The YouTube video at the right gives an impression of Indiegame Garden.

Download (version Beta-6)
Options:
  1. Internet Explorer only - run the ClickOnce web installer by clicking this link
  2. Download the installer here and run it. The installer downloads the software and should also install any required Microsoft libraries (such as XNA 4.0).
  3. For Portable App fans, download the ZIP file here; unzip somewhere for example the Desktop; and run IndiegameGarden.exe.


System requirements
  • Windows XP or Windows 7 PC with somewhat modern graphics card [XNA "Reach" profile, for the insiders.]
  • Fast Internet connection (for the automatic game downloads and web install program)
  • For the Portable App version only: on Windows 7 you may need to install the  Microsoft XNA 4.0 redistributable. (You can first try without installing this.) On Windows XP you may need to install the  Microsoft XNA 4.0 redistributable, DotNet 4.0 or in rare cases DirectX. (You can first try without installing this.)
Play
Enable your internet connection. Run Indiegame Garden, wait a moment (first time may be longer), and check the panel that appears on the top left which keys to use. Have fun! Future updates will automatically show in the form of an update icon top left in your garden.
